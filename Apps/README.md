# Apps

Public notes for app concepts inspired by WWDC26.

Do not put proprietary app source code here. Real product apps should live in private app repositories by default.

Use this folder for public-facing material only:

- App idea summaries.
- Build logs that are safe to share.
- Screenshots or videos meant for marketing.
- Links to extracted demos in `Demos/`.
- Notes about which WWDC26 technologies inspired the idea.

Small technical experiments should still start in `Demos/`.
