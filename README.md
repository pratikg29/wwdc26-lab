# wwdc26-lab

Practical SwiftUI and Apple platform experiments from WWDC26.

## Purpose

This repo is a public lab for learning, testing, and sharing new Apple platform ideas from WWDC26. The focus is small working demos, clear notes, and practical takeaways that can help iOS and SwiftUI developers move from announcement to implementation.

## What goes here

- Small SwiftUI and Apple platform demos.
- Notes from Apple sessions, documentation, and sample code.
- App experiments inspired by WWDC26 technologies.
- Reusable packages only when a demo becomes useful beyond the lab.

## Structure

```text
Notes/       Public notes from Apple sessions and docs
Demos/       Small focused experiments
Apps/        App prototypes and larger experiments
Packages/    Reusable Swift packages split from demos
Resources/   Public screenshots, recordings, and supporting assets
Templates/   README and demo templates
```

## Demo format

Each demo should include:

- What this demonstrates.
- Official source links.
- What I learned.
- How to run it.
- Screenshots or a short recording when useful.
- Where the idea could fit in a real app.

## Source policy

Official Apple documentation, sessions, release notes, and sample code are the source of truth for factual claims. Community discussion can inspire ideas, but it should not be treated as confirmation.
